# Parastu Kandi Promo Page

This repository hosts a lightweight web page that plays a short promotional audio for **Parastu Kandi**. The page is designed for easy sharing via QR code and works reliably across modern browsers.

## Features
- Plays a short promo audio when the page is visited (with fallback for browsers that block autoplay).
- Minimalist, fast-loading page.
- Supports multiple promo audios selectable via URL query parameters: `?promo=1, ?promo=2`
- Designed to be shared using GitHub Pages → Bit.ly → QR code workflow.

### Notes
- Some browsers may block autoplay. The page includes a simple fallback overlay requiring a click/tap.
- Designed for easy sharing via QR code or short links.
